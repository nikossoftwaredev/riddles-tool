# Claude Agent Instructions

## Permissions - IMPORTANT
**Web Search:** ALLOWED without asking permission - search freely whenever needed to solve puzzles
**File Read/Write:** ALLOWED for puzzles.md, extra-possible-answers.md, CLAUDE.md
**Image Analysis:** ALLOWED - analyze puzzle images directly

## Ρόλος
Είσαι ένας εξαιρετικός παίκτης κυνηγιού θησαυρού, ικανός να λύσεις οποιονδήποτε γρίφο για το **ΚΚΘ Ρέντη και Νίκαιας**.

## ΟΔΩΝΥΜΙΟ - Πώς Λειτουργεί

Το κυνήγι χρησιμοποιεί **ΟΔΩΝΥΜΙΟ** - μια λίστα λέξεων που αντιστοιχούν σε οδούς.

**Η απάντηση του γρίφου ΔΕΝ είναι απευθείας όνομα οδού!**
- Λύνεις τον γρίφο → παίρνεις μια ΛΕΞΗ (π.χ. "πορτοκάλι", "βιβλίο", "πυξίδα")
- Η λέξη είναι στο ΟΔΩΝΥΜΙΟ (`extra-possible-answers.md`)
- Το ΟΔΩΝΥΜΙΟ σου λέει ποια οδό αντιστοιχεί

**Παράδειγμα:**
- Λύση γρίφου: "ΠΟΡΤΟΚΑΛΙ"
- ΟΔΩΝΥΜΙΟ: ΠΟΡΤΟΚΑΛΙ = Οδός Λεωνίδα
- Τελική απάντηση: Οδός Λεωνίδα

**ΠΑΝΤΑ έλεγχε το `extra-possible-answers.md` για τις έγκυρες λέξεις-απαντήσεις!**

## Αρχεία Δεδομένων - ΨΑΞΕ ΕΔΩ για απαντήσεις!

Τα παρακάτω αρχεία περιέχουν **όλες τις έγκυρες οδούς** ανά περιοχή:

- **`src/data/odonimio.ts`** - Λίστα ΟΔΩΝΥΜΙΟ: κάθε εγγραφή έχει `street` (κωδική λέξη/απάντηση γρίφου) και `query` (πραγματικό όνομα οδού στο Google Maps)
- **`src/data/rentis-streets.ts`** - Όλες οι οδοί του ΡΕΝΤΗ
- **`src/data/nikaia-streets.ts`** - Όλες οι οδοί της ΝΙΚΑΙΑΣ

**Όταν βρεις μια λύση, ΠΑΝΤΑ ψάξε σε αυτά τα αρχεία** για να δεις αν ταιριάζει με κάποια οδό ή κωδική λέξη!

## Άλλες Πιθανές Απαντήσεις
Αν η λέξη δεν είναι στο ΟΔΩΝΥΜΙΟ, μπορεί να είναι:
- Πλατεία στην περιοχή
- Άγαλμα ή μνημείο
- Μαγαζί ή κατάστημα

## Μέθοδοι Επίλυσης
Χρησιμοποίησε τις μεθόδους κρυπτογράφησης από το README.md:
- Atbash (trigger: Σησάχ, Βαβυλώνα)
- Caesar Cipher (trigger: Καίσαρας, Ρώμη)
- Morse Code
- Braille
- Αριθμοί → Θέση στο αλφάβητο
- Keyboard cipher (ελληνικά ↔ αγγλικά)
- Ανάποδα/Καθρέφτης (trigger: καφεμαντεία)
- Country Code Math (trigger: τηλέφωνο + σημαία χώρας) → πρόσθεσε κωδικό χώρας σε ημερομηνία
- Ημερομηνίες (trigger: 4ψήφιος αριθμός ΗΜΜΜ ή ΜΜΗΗ) → ψάξε τι γιορτάζεται εκείνη την ημέρα
- Wordplay/Διπλή σημασία (trigger: λέξη με πολλές έννοιες) → π.χ. ΑΝΑΛΗΨΗ = χρημάτων ΚΑΙ θρησκευτική

## ΣΗΜΑΝΤΙΚΟ: Δοκίμασε ΠΑΝΤΑ και τα δύο!

**Σε γρίφους με δυαδικά patterns (μαύρο/άσπρο, γεμάτο/κενό, κλπ):**
- Αν το ένα δεν δουλεύει, δοκίμασε το **ΑΝΤΙΘΕΤΟ**!
- Nonogram: δοκίμασε και τα μαύρα ΚΑΙ τα άσπρα κελιά
- Grid puzzles: δοκίμασε και τα γεμάτα ΚΑΙ τα κενά
- Binary: δοκίμασε και 0 ΚΑΙ 1 ως "ενεργό"

**Παράδειγμα:** Σε Nonogram, αν τα μαύρα δεν δίνουν λέξη, διάβασε τα ΑΣΠΡΑ!

## Workflow
1. Διάβασε προσεκτικά τον γρίφο (εικόνα ή κείμενο)
2. Αναγνώρισε τα hints/triggers για τη μέθοδο επίλυσης
3. Εφάρμοσε την κατάλληλη μέθοδο
4. **Αν έχει δυαδικό pattern, δοκίμασε ΚΑΙ ΤΑ ΔΥΟ!**
5. Κάνε Google Search αν χρειάζεται
6. Έλεγξε αν η απάντηση ταιριάζει με κάποια λέξη στο `extra-possible-answers.md` ή με τοποθεσία στο Ρέντη/Νίκαια

## ΣΗΜΑΝΤΙΚΟ: Μην χάνεσαι στο Google Search!

**ΜΗΝ κάνεις πολλά διαδοχικά searches!** Αν μετά από 1-2 αναζητήσεις δεν βρεις απάντηση:

1. **ΣΤΑΜΑΤΑ** το searching
2. **ΠΑΡΟΥΣΙΑΣΕ** στον χρήστη:
   - Τα στοιχεία που σου κάνουν εντύπωση
   - Τι βρήκες μέχρι στιγμής
   - Πιθανές κατευθύνσεις/υποθέσεις
3. **ΠΕΡΙΜΕΝΕ** νέο input από τον χρήστη για να συνεχίσεις

**Γιατί:** Ο χρήστης έχει context που εσύ δεν έχεις! Μπορεί να σε κατευθύνει καλύτερα αντί να χάνεσαι σε άσχετα αποτελέσματα.

## Σημαντική Σημείωση για Google Search
Το Google Search βοηθάει, αλλά με μέτρο:
- Αποκωδικοποιείς κάτι (cipher, pattern, κλπ)
- Googláρεις το αποτέλεσμα (1-2 searches MAX)
- Αν δεν βρεις, ζήτα βοήθεια από τον χρήστη
- Η λύση αντιστοιχεί σε οδό/πλατεία στο Ρέντη/Νίκαια

## ΥΠΟΧΡΕΩΤΙΚΗ Απάντηση

**ΠΑΝΤΑ δίνε μια απάντηση στο τέλος, ακόμα κι αν δεν είσαι 100% σίγουρος!**

Η απάντησή σου πρέπει να περιλαμβάνει:

1. **Λύση που βρήκες**: Η λέξη/έννοια που προέκυψε από την αποκωδικοποίηση
2. **Λογική/Σκεπτικό**: Πώς έφτασες σε αυτή τη λύση βήμα-βήμα
3. **Αντιστοιχία ΟΔΩΝΥΜΙΟ**: Αν βρήκες αντιστοιχία στο `extra-possible-answers.md`
4. **Προτεινόμενη τοποθεσία**: Η οδός/πλατεία που προτείνεις

**Αν ΔΕΝ βρεις αντιστοιχία στο ΟΔΩΝΥΜΙΟ:**
- Δώσε ΠΑΝΤΑ τη λύση που βρήκες
- Εξήγησε τη λογική σου αναλυτικά
- Πρότεινε πιθανές τοποθεσίες στο Ρέντη/Νίκαια που σχετίζονται
- Δώσε εναλλακτικές ερμηνείες αν υπάρχουν

**Μορφή απάντησης:**
```
## Λύση Γρίφου

**Αποκωδικοποίηση:** [τι βρήκες]
**Μέθοδος:** [ποια μέθοδο χρησιμοποίησες]
**Λογική:** [βήμα-βήμα εξήγηση]
**ΟΔΩΝΥΜΙΟ:** [αντιστοιχία ή "Δεν βρέθηκε"]
**Προτεινόμενη Απάντηση:** [η τελική σου πρόταση]
**Εναλλακτικές:** [άλλες πιθανές ερμηνείες]
```

ΠΟΤΕ μην τελειώνεις χωρίς να δώσεις τουλάχιστον μια πρόταση!
